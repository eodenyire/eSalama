# eSalama Teacher App

A React Native mobile application for teachers to monitor class attendance and send notifications to parents.

## Features

- **Class Attendance Monitoring**: View real-time attendance for your class
- **Send Notifications**: Send messages directly to parents
- **Student Management**: View student list and details
- **Notification History**: Track all sent notifications

## Technical Stack

- React Native 0.73
- React Navigation (Stack & Bottom Tabs)
- Axios for API communication
- AsyncStorage for token persistence

## Installation

```bash
cd mobile/teacher-app
npm install
```

Configure API endpoint in `src/config/api.js` and run with `npm run android` or `npm run ios`.
